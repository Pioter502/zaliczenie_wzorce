Wzorzec kreacyjny o nazwie Budowniczy daje możliwość tworzenia złożonych obiektów etapami.
Pozwala on produkować różne typy oraz reprezentację obiektu używając tego samego kodu konstrukcyjnego.
Jego głównym zastosowaniem jest chęć pozbycia się długich konstruktorów. Wywołanie konstruktora z np.: 10 parametrami to niefajna sprawa.
Dodatkowo można go stosować gdy potrzebujemy różnej reprezentacji jakiegoś obiektu.
Przykładem może być mój kod tworzący wybraną pizze.
